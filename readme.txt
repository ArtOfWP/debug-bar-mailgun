Adds a new panel for MailHog to Debug Bar. Alpha version.

== Description ==
The plugins adds a new panel to Debug Bar where you can interact with MailHog. Currently it only supports listing emails. To enable MailHog on your local development environment see https://github.com/mailhog/MailHog
To enable it go to Tools->Mailhog and enter the values that corresponds to your MailHog setup.

== Installation ==
Install via default plugin installer. To enable it go to Tools->Mailhog and enter the values that corresponds to your MailHog setup.
